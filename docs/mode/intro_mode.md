# イントロモード

`02030904` からの 4バイトは、イントロ(起動時〜タイトル画面)でのモードを表しています。

```
0x083856f8
  00: ???
  01: ???
  02: ???
  03: ???
  04: タイトル画面 (0x08385e10)
    03: タイトル画面
      04: タイトル画面でユーザーの入力受付 (0x080ec35c)
        Up:      カーソルを上に
        Down:    カーソルを下に
        A,Start: 次へ
      05: 32フレーム待って、次(0x404)へ
    04: プレイヤーのタイトル画面のアクションに対応
      TitleCursorIdx(0x02030b38)が
        00: ゲームモードを はじめからモード へ & イントロモードを 0x5 へ
        01: ゲームモードを 続きからモード へ & イントロモードを 0x5 へ
        02: ミニゲーム(0x6)へ
  05: ゲーム(MainProcess) に移行(ゲームモードは直前に設定済み)
  06: ミニゲーム
```
